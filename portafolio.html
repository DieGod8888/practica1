Portafolio

Aplicación Web: Gestor de Tareas

Desarrollé una aplicación web para la gestión de tareas utilizando React y Node.js. La aplicación permite a los usuarios crear, editar y eliminar tareas, así como marcarlas como completadas. También incluye autenticación de usuarios y almacenamiento en la nube.

  • Tecnologías: React, Node.js, MongoDB, Express
  • Fecha: Enero 2023

Tienda en Línea: Moda Urbana

Creé una tienda en línea para una marca de moda urbana utilizando WordPress y WooCommerce. La plataforma incluye un catálogo de productos, carrito de compras, integración con pasarelas de pago y un panel de administración para gestionar inventario y pedidos.

  • Tecnologías: WordPress, WooCommerce, PHP, CSS
  • Fecha: Agosto 2022

Sitio Web Corporativo: Consultoría IT

Diseñé y desarrollé un sitio web corporativo para una empresa de consultoría en TI. El sitio incluye información sobre servicios, portafolio de clientes, blog y formulario de contacto. Fue desarrollado con HTML, CSS y JavaScript, y es totalmente responsive.

  • Tecnologías: HTML, CSS, JavaScript, Bootstrap
  • Fecha: Marzo 2022

Aplicación Móvil: Clima en Tiempo Real

Colaboré en el desarrollo de una aplicación móvil que muestra el clima en tiempo real utilizando una API de pronóstico del tiempo. La aplicación fue desarrollada con React Native y ofrece información detallada sobre temperatura, humedad y condiciones climáticas.

  • Tecnologías: React Native, API de Clima, Firebase
  • Fecha: Noviembre 2021
